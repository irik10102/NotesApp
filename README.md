# 📝 NotesApp

A full-stack CRUD (Create, Read, Update, Delete) Notes Taking Application built with **React.js**, **Node.js**, and **Express.js**. This application allows users to create, view, update, and delete notes through an intuitive user interface.

---

## 🚀 Features

* Create new notes
* View all notes
* Update existing notes
* Delete notes
* Responsive user interface
* RESTful API integration

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

## ⚙️ Installation & Setup

Follow the steps below to run the project on your local machine.

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate to the Project Directory

```bash
cd NotesApp
```

### 3. Install Frontend Dependencies

Navigate to the frontend folder:

```bash
cd NotesAppFrontend
```

Install all required frontend packages:

```bash
npm install
```

### 4. Install Backend Dependencies

Move to the backend folder:

```bash
cd ../NotesAppBackEnd
```

Install all required backend packages:

```bash
npm install
```

### 5. Start the Backend Server

Run:

```bash
npm start
```

The backend server will start and listen on its configured port.

### 6. Open a New Terminal

Open a new terminal in VS Code (or your preferred code editor).

### 7. Start the Frontend Application

If you opened the project root (`NotesApp`) in your editor, navigate to the frontend directory:

```bash
cd NotesAppFrontend
```

Start the frontend development server:

```bash
npm run dev
```

### 8. Open the Application

Visit:

```text
http://localhost:5173
```

in your browser.

### 9. Interact with the Application

You can now:

* ➕ Create notes
* 📖 View notes
* ✏️ Edit notes
* 🗑️ Delete notes

Enjoy using the NotesApp!
